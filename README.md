# **OpenProtocols | PACK (v2025.1)**

## **Repository Overview**

This repository hosts the **Preservation & Access Compliance Kit (PACK)**.

These Standard Operating Procedures (SOPs) are designed to provide a turnkey compliance solution for:

* **NIH Policy NOT-OD-21-013:** Data Management and Sharing (DMS) Policy.  
* **NIH Notice NOT-OD-21-016:** Selecting a Repository for Data Resulting from NIH-Supported Research.  
* **OSTP 2022 Memorandum:** Ensuring Free, Immediate, and Equitable Access to Federally Funded Research.

## **Deployment**

This site is built using single-file HTML5 with embedded Tailwind CSS for portability. It is hosted via GitHub Pages to ensure high availability and version persistence.

## **Usage**

Principal Investigators and Lab Managers may fork this repository or download PACK\_SOP.md to include in internal laboratory handbooks.
